For Technical Issues, Please Include:

  • Browser name and version (Chrome 91, Firefox 89, etc.)
  • Operating system (Windows 10, macOS Big Sur, etc.)
  • Device type (desktop, mobile, tablet)
  • Specific calculator experiencing issues
  • Error message text (if any)
  • Steps to reproduce the problem
  • Screenshots or screen recordings (if helpful)

For Calculator Questions, Please Specify:

  • Which calculator you're using
  • Input values you're entering
  • Expected vs. actual results
  • Your specific use case or context
  • Any formulas or methodologies you're comparing against

For Feature Requests, Please Describe:

  • What functionality you'd like to see added
  • How it would improve your workflow or experience
  • Similar tools or features you've seen elsewhere
  • Your background or use case for the feature
  • Priority level for your needs
